# Subscription Manager

A React application to manage your monthly subscriptions.

## Setup

Since the automated setup encountered issues, please run the following commands manually:

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Features

- **Dashboard**: View all your subscriptions and total monthly cost.
- **Add Subscription**: Easily add new subscriptions with cost and billing cycle.
- **Tracking**: See days remaining until next payment.
- **Persistence**: Data is saved to your browser's local storage.

## Tech Stack

- React + Vite
- Vanilla CSS (Premium Dark Theme)
